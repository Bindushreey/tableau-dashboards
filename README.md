# Tableau Dashboard Portfolio

This repository contains Tableau dashboards created as part of data analytics projects.

## Projects Included

### 1. Bank Customer Churn Analysis
An interactive dashboard analyzing customer churn patterns using KPIs and segmentation.

Key insights:
- Churn rate: 14.22%
- Churn higher in specific age groups
- Credit score impacts churn probability
- Tenure influences retention

### 2. Logistics Delay Analysis (In Progress)
Dashboard exploring relationship between waiting time and shipment delays.
